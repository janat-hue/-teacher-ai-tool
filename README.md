# Teacher AI Task Design Tool

Jednoduchá webová aplikace pro učitele, která pomáhá navrhovat úkoly rozvíjející myšlení žáků v době generativní AI.

Aplikace vychází z modelu:

OVU → H1–H3 → AIAS → OSAP → AI stopa → hodnocení → reflexe

## Cíl projektu

Pomoci učitelům:

- navrhovat úkoly, které rozvíjejí myšlení
- zabránit jednoduchému outsourcingu úkolů na AI
- sledovat proces myšlení žáků, nejen výsledný produkt
- pracovat s reflexí a formativním hodnocením

## Hlavní části aplikace

1. Proč některé úkoly selhávají v době AI  
2. Vysvětlení modelu  
3. Aktivita: rozpoznání slabého zadání  
4. Aktivita: redesign zadání  
5. Generátor vlastního zadání  

## Použité technologie

- HTML
- CSS
- JavaScript

Aplikace běží čistě na frontendu bez backendu.

## Jak aplikaci spustit

1. Stáhni repozitář
2. Otevři soubor `index.html` v prohlížeči
